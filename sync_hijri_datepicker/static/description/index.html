Web Hijri(Islamic) Datepicker

This module is used to select date according to Islamic Hijri months. Normal odoo datepicker is also available with Hijri date picker. you can select date from any of the picker and it will be converted vice-versa in all modules

This module is compatible with

  • Community
  • Enterprise
  • Odoo.sh

Key Features

    If you choose Hijri date, the module will reflect automatic Greogorain Calender and vice-versa.
    This Date picker addons you can apply with any existing or new object for odoo.
    You can see the Hijri Calender and Greogorain Calender on the same screen.

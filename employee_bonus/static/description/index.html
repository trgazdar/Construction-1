Employee Bonus

Employee Bonus Management in Odoo

This app allows your employees to create bonus requests in Odoo and send for Department manager and HR Manager approval.

Main Features:

  • Manager can configure the bonus reasons for giving the bonus.
  • Employees can create the Bonus Request in Odoo.
  • Bonus reasons can be selected during creating employee bonus requests.
  • Department Manager can see Bonuses To Approve by Department Menu and Approve Bonus request by Employee.
  • HR Manager can see Bonuses To Approve by Officer Menu and Approve Bonus by Employee.
  • Print Employee Bonus PDF Report.
